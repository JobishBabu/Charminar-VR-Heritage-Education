# Charminar VR – Immersive Heritage Education Experience

## Project Overview
This project presents an immersive Virtual Reality walkthrough of Charminar developed using Unity 2022 LTS and XR Interaction Toolkit.

## Features
- Real-scale 3D simulation
- Teleport and Snap Turn locomotion
- Educational Hotspots
- Spatial Audio Narration
- Artifact Interaction
- Optimized for Meta Quest (≥72 FPS)

## Tools Used
- Unity 2022 LTS
- XR Interaction Toolkit
- Blender
- OpenXR
- C#

## Deployment
- Meta Quest APK
- PC VR Executable

## Performance Metrics
- Quest FPS: 78–85
- PC VR FPS: 95
